#游戏运行步骤
## 第一步，打开网页，可以看到敌人(小虫)，已经开始以不同速度，行动起来。

## 第二步，使用键盘上的上下左右按键控制，玩家上下左右移动躲避敌人，并成功到达水域。

### 如果玩家与敌人相撞，则玩家回到起始位置，游戏重新开始。
### 如果玩家移动超出左右下游戏区域时，玩家回到起始位置，游戏重新开始。

## 第三步，玩家成功到达水域后，游戏胜利并提示。
